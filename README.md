# Desert-Storm-NGC
Sistema di gestione strategica Tempesta del Deserto per battaglie alleanza. Mappa interattiva degli 11 edifici, assegna operatori illimitati a Kill Swat e ogni edificio, visualizza strategia ottimizzata, cattura screenshot per condividere. Salvataggio automatico locale, ottimizzato mobile. Coordina perfettamente gli attacchi alleanza!
